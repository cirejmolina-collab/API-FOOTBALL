# API Football Explorer

A simple web app to search for football leagues and cups using the API-Football service.

## Usage
- Enter a league or cup name and click Search.
- Displays league/cup info, country, season, and logo.

## Setup
- Replace `YOUR_API_KEY_HERE` in script.js with your API-Football key.
- Requires a free API key from https://dashboard.api-football.com/
